How to Create Your Own Website with HTML and CSS

Learn to build and style your first website from scratch

What is HTML and CSS?

HTML provides the structure of a webpage, allowing you to add elements like headings, paragraphs, lists, images, videos, and links. CSS enhances the page by styling and customizing elements, defining colors, fonts, and layouts.

Step 1 – Set Up Your Text Editor and HTML File

To begin, you'll need a text editor for writing code. Options include Notepad++ for Windows or Sublime Text for Mac. Create a new file called index.html inside a folder named "Your first web page".

Start by adding these fundamental elements:

  1. DOCTYPE declaration
  2. HTML tags
  3. Language attribute
  4. Head and body sections

Example of an HTML Document

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Example HTML Document</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is an example of an HTML document.</p>
</body>
</html>

Additional Elements to Include

You can add more structure and interactivity using:

  • Headings and paragraphs
  • Lists (ordered and unordered)
  • Images and videos
  • Links to other pages or websites

Styling Your Website with CSS

CSS (Cascading Style Sheets) lets you define visual styles for your HTML elements. Here's a simple example:

body {
  font-family: Arial, sans-serif;
  background-color: #f4f4f4;
}
h1 {
  color: #333;
}
p {
  color: #666;
}

Example of an HTML Document with CSS

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Styled HTML Document</title>
  <style>
    body {
      font-family: Arial, sans-serif;
      background-color: #f4f4f4;
    }
    h1 {
      color: #333;
    }
    p {
      color: #666;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is an example of an HTML document with CSS.</p>
</body>
</html>
